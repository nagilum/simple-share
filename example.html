Simple Share

There are a few ways to use the Simple Share plugin. Just give an element the class simple-share and you're mostly there.

Example #1: Providing URL to Share

The following code will produce buttons to share the URL https://github.com/ to both Facebook and Twitter when pressed. 

      
      <div class="simple-share" data-url="https://github.com/" data-networks="facebook,twitter"></div>

Example #2: Omitting the URL

If you don't provide the data-url attribute in the element tag, the current window URL will be used. 

      
      <div class="simple-share" data-networks="facebook,twitter"></div>

Example #3: Omitting the Networks

You can even omit the data-networks attribute, in which case all available networks the plugin supports will be added. 

      
      <div class="simple-share"></div>

Other Options

Other attributes options you can add to the element are:

  • data-no-icons - this will explude the icon from being added.
  • data-add-text - this will add the title for each network being added.
      
      <div class="simple-share" data-add-text data-no-icons data-networks="facebook"></div>

Supported Networks

  • Facebook
  • reddit
  • Twitter
